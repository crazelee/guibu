复合数据类型

2018/11/12 posted in  Golang入门

数组

数组长度是固定的。

var a [3]int  // array of 3 integers
var b [3]int = [3]int{1,2,3}
c := [...]int{1, 2, 3}

如果两个数组的所有元素都相等，数组相等

a := [2]int{1, 2}
b := [...]int{1,2}
fmt.Println(a==b)  // true

切片 Slice

变长的序列，每个元素有相同的类型。
一般写作[]T，Slice的语法和数组很像，只是没有固定长度而已。

一个slice由三部分构成：指针、长度、容量。内置的len和cap函数返回对应长度和容量。
len(s) == 0判断是否为空，不应该用 s == nil

哈希表 Map

args := make(map[string][int])  // 创建key为string，value为int型


# 初始化 1
ages := map[string][int]{
    "li" : 31,
    "bob" : 23,
}

相当于

# 初始化 2
ages := make(map[string][int])
ages["li"] = 31
ages["bob"] = 23

delete删除元素

delete(ages, "li") // remove element ages["li"]

把忽略value的map当做一个字符串集合，map[string]bool

结构体

type Employee struct {
    ID int
    Name, Adress string
    Salary int
}

var dilbert Employee
dilbert.Name   // 访问成员变量

如果结构体成员名字是以大写字母开头的，那么该成员就是导出的
考虑效率的话，作为函数的变量，较大的结构体通常会用指针的方式传入和放回

结构体嵌入和匿名成员

type Point struct {
    X, Y int
}

type Circle struct {
    Center Point
    Redius int
}

type Wheel struct {
    Circle Circle
    Spokes int
}

访问会变得繁琐

var w Wheel
w.Circle.Center.X = 8
w.Circle.Center.Y = 8
w.Circle.Radius = 5
w.Spokes = 20

Go语言的特性：可以只声明一个成员对应的数据类型而不指定成员名

type Point struct {
    X, Y int
}

type Circle struct {
    Point
    Redius int
}

type Wheel struct {
    Circle 
    Spokes int
}

var w Wheel
w.X = 8
w.Y = 8
w.Radius = 5
w.Spokes = 20
